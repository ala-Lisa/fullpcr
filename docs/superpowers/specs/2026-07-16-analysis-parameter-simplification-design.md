# 分析参数区域精简与帮助说明设计

## 目标

- 从网站参数区域删除“从输入文件同步路径”、引物类型预设和“应用参数预设”。
- 基础参数始终显示；高级参数默认收起，由用户按需展开。
- 将 `bind` 和 `cutprimer` 两个输出开关并入高级参数。
- 为全部基础参数和高级参数提供准确的问号说明，写明应用、真实参数名、用途、单位及默认行为。
- 不删除后端参数构建能力，不改变五步分析命令语义、核心算法或 CLI。

## 页面结构

外层 `分析参数` 区域保持现有位置，内部依次显示：

1. `基础参数`
2. `显示高级参数` 开关，默认关闭
3. 开启时显示高级数值参数以及 `bind`、`cutprimer`
4. 现有命令预览控制

路径同步和参数预设只从 GUI 移除。输入验证后的项目路径仍按现有自动规则衔接工作流；后端预设辅助函数暂时保留，避免扩大接口变更范围。

## 参数与帮助说明

### 基础参数

- 最小扩增片段长度：`MFEprimer spec · -s`
- 最大扩增片段长度：`MFEprimer spec · -S`
- 允许错配数：`MFEprimer spec · --misMatch`
- obipcr 错配等级：`fullpcr run / obipcr · --mismatches`
- 环状序列：`fullpcr run / obipcr · --circular`

### 高级参数

- 最低 Tm：`MFEprimer spec · -t`
- 错配区域起点：`MFEprimer spec · --misStart`
- 错配区域终点：`MFEprimer spec · --misEnd`
- 单价离子浓度：`MFEprimer spec · --mono`
- 二价离子浓度：`MFEprimer spec · --diva`
- dNTP 浓度：`MFEprimer spec · --dntp`
- 引物浓度：`MFEprimer spec · --oligo`
- 输出引物结合位点：`MFEprimer spec · -b / --bind`
- 从扩增序列中切除引物：`MFEprimer spec · --cutprimer`

帮助文字采用“应用/模块 + 实际参数名 + 作用 + 默认行为”的固定结构。高级数值参数未启用时继续沿用现有默认命令；折叠高级区域只隐藏控件，不改变已经启用的参数和值。

## 状态规则

- `显示高级参数` 仅控制可见性，不负责启用或禁用参数。
- 展开后勾选的高级参数在再次收起时继续生效。
- 重新展开时恢复此前的启用状态和值。
- 页面切换、输入验证和高级分步工作流开关不得清除这些状态。
