# 工作台开关间距设计

## 目标

进一步压缩“显示高级参数”和“显示高级分步工作流”附近的纵向空白，并删除截图中两条多余横线。

## 实现范围

- 删除 `_render_analysis_parameter_controls()` 返回参数前的 `st.divider()`。
- 删除 `_render_quick_analysis()` 标题前的 `st.divider()`。
- 继续使用分析工作台已有的 `0.72rem` 专属垂直间距，让两个开关和快速分析自然紧凑排列。
- 不缩小开关、字体、帮助图标或点击区域。

## 行为保持

高级参数展开、高级分步工作流显示、参数状态保存、快速分析和命令构建均保持不变。

## 页面隔离

只删除上述两个函数中的指定分隔线。其他页面和高级工作流内部的分隔线不受影响。

## 测试与验收

- 源码测试确认两个指定位置不再调用 `st.divider()`。
- 既有高级参数、高级工作流和快速分析 AppTest 继续通过。
- 运行 GUI 测试、全量测试、编译检查和 `git diff --check`。

## 范围外

不修改其他横线、控件尺寸、颜色、页面结构、参数或后端分析流程。
